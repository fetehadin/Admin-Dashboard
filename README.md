# Admin Dashboard

A modern admin dashboard built with HTML and CSS as part of the Intermediate HTML and CSS course from **[The Odin Project](https://www.theodinproject.com)**.

## Overview

This project focuses on building a complete dashboard. The dashboard includes a sidebar, header, project cards, announcements, and trending sections.

## Features

- Full dashboard layout using CSS Grid
- Nested grid containers
- Sidebar navigation
- Search bar and user profile section
- Project cards layout
- Announcements and trending panels
- Modern and clean UI design
- SVG icons integration

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Google Fonts
- Material Design Icons

## What I Learned

- Structuring large layouts with Grid
- Combining Grid and Flexbox effectively
- Creating reusable UI components
- Improving layout organization and spacing
- Building professional dashboard interfaces
